# TvShow
Welcome to TV series app! The app is built in MVVM architecture &amp; RxJava library.For #ui layer change it integrates with Live data observer and for business logic, it integrates with rxJava.The app also has a Room database for storing watched series. Pagination has been used with Recyclerview  &amp; Retrofit is for fetching data
